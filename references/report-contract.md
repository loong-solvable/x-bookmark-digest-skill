# Report Contract

The agent should write these two files after export:

## `report/bookmark-digest.md`

Recommended sections:

1. Overview
2. What This Bookmark Library Is Really About
3. Main Categories
4. Read First
5. Repeated Authors / Domains / Themes
6. Worth Acting On Now
7. Worth Archiving As Reference
8. Suggested Cleanup Rules

## `report/bookmark-index.md`

Recommended format:

- Group by category
- Each bookmark entry includes:
  - short title
  - one-line summary
  - author
  - link
  - optional note: actionable / reference / inspiration / noise

## Tone

- Crisp
- High signal
- No filler
- Explain category names in plain language

## Important

- Avoid category names that are too generic like "Misc"
- If something is genuinely mixed, explain why it was grouped that way
- Preserve original X links whenever possible
